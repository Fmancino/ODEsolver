Created on: Dic, 2015 

Author: Francesco Mancino

Description: Abstract Base class for the family of methods for solving ODE of the form dy/dt=f(y,t). The possible methods are defined as derived classes from this class.

Virtual function the solves the ODE with initial conditions, Time interval, and number of steps specified in the methods of "AbstactOdeSolver". The first input is the right hand side of the ODE, also a class, and the second input is the file in which the solution will be saved.

Implemented in BackwardEulerSolver, ForwardEulerSolver, FourStepSolver, RungeKutta4Solver, TwoStepSolver, and ThreeStepSolver.
